Coding "environment" for JS

This has inspirations from Codepen and Jupyter Notebook, only in this case we use it for JavaScript. Users can write code and run in on the spot. Importing of various modules from NPM is also supported. One can write React code as well. Basically, it has all the writing and running aspect of writing code. Features like saving and multiple tabs are planned but not yet implemented. Also, there's no server involved. Everything happens on users' browser. Yes, even the compiling.

I did this immediately after learning TypeScript and as such employed its features as much as possible. Unlike with JS, this time the development process was not so much about referencing the code repeatatively. With a little more work, TS certainly eased the process. I can say honestly that I enjoyed writing the code for the project.

The live demo is hosted by Vercel. Here is the repo.
